# 演讲评价标准

## 概述

本文档定义AIMC数字人主持系统中演讲评价的维度、标准和评分方法。

## 评价维度

### 1. 内容质量 (Content Quality)

权重：30%

#### 1.1 表达清晰度 (Clarity)
**评分范围**：0-10分

**评分标准**：
- **9-10分**：表达极为清晰，逻辑严密，没有任何歧义
- **7-8分**：表达清晰，逻辑连贯，易于理解
- **5-6分**：表达基本清楚，逻辑尚可，但有少量模糊之处
- **3-4分**：表达不够清晰，逻辑混乱，难以理解
- **0-2分**：表达非常模糊，完全无法理解

**检查要点**：
- 主题句是否明确
- 逻辑层次是否清晰
- 是否避免了歧义表达
- 专业术语是否解释清楚

#### 1.2 洞见深度 (Insight)
**评分范围**：0-10分

**评分标准**：
- **9-10分**：观点极具创新性和启发性，让人眼前一亮
- **7-8分**：观点有深度，有独特见解
- **5-6分**：观点基本合理，但缺乏新意
- **3-4分**：观点较为平庸，没有新意
- **0-2分**：观点陈腐，毫无价值

**检查要点**：
- 是否有新的观点
- 是否有独特的视角
- 是否能引发思考
- 是否有实用价值

#### 1.3 内容丰富度 (Richness)
**评分范围**：0-10分

**评分标准**：
- **9-10分**：内容非常丰富，涵盖多方面，信息量大
- **7-8分**：内容充实，有足够的细节和案例
- **5-6分**：内容基本完整，但不够丰富
- **3-4分**：内容单薄，缺乏细节和案例
- **0-2分**：内容空洞，几乎没有实质内容

**检查要点**：
- 是否有足够的案例
- 是否有数据支撑
- 涵盖面是否广泛
- 是否有深度

#### 1.4 可信度 (Credibility)
**评分范围**：0-10分

**评分标准**：
- **9-10分**：极其可信，数据权威，来源可靠
- **7-8分**：比较可信，有数据支撑，来源明确
- **5-6分**：基本可信，但数据或来源不够明确
- **3-4分**：可信度较低，缺乏数据支撑
- **0-2分**：完全不可信，没有任何依据

**检查要点**：
- 是否有数据来源
- 数据是否权威
- 观点是否有依据
- 是否有逻辑漏洞

---

### 2. 表达能力 (Delivery)

权重：25%

#### 2.1 流畅度 (Fluency)
**评分范围**：0-10分

**评分标准**：
- **9-10分**：极为流畅，没有任何卡顿，一气呵成
- **7-8分**：流畅自然，仅有极少停顿
- **5-6分**：基本流畅，但有一些停顿和重复
- **3-4分**：不够流畅，卡顿较多，影响理解
- **0-2分**：非常不流畅，几乎无法连续表达

**检查要点**：
- 是否有重复
- 是否有卡顿
- 是否有口误
- 整体是否连贯

#### 2.2 语速控制 (Pace)
**评分范围**：0-10分

**评分标准**：
- **9-10分**：语速完美，节奏掌控极佳
- **7-8分**：语速适中，节奏良好
- **5-6分**：语速基本适中，但有时过快或过慢
- **3-4分**：语速过快或过慢，影响理解
- **0-2分**：语速完全不当，无法理解

**检查要点**：
- 语速是否适中
- 节奏是否有变化
- 是否能根据内容调整语速
- 关键内容是否放慢

#### 2.3 语气表达 (Tone)
**评分范围**：0-10分

**评分标准**：
- **9-10分**：语气极为贴切，完美表达情感
- **7-8分**：语气贴切，能很好表达情感
- **5-6分**：语气基本合适，能表达基本情感
- **3-4分**：语气不够贴切，情感表达不足
- **0-2分**：语气完全不当，没有情感表达

**检查要点**：
- 语气是否与内容匹配
- 是否有情感变化
- 是否能感染听众
- 是否自然

#### 2.4 感染力 (Charisma)
**评分范围**：0-10分

**评分标准**：
- **9-10分**：极具感染力，完全吸引听众
- **7-8分**：有感染力，能吸引听众
- **5-6分**：有一定感染力，能维持听众注意力
- **3-4分**：感染力较弱，难以吸引听众
- **0-2分**：完全没有感染力，听众失去兴趣

**检查要点**：
- 是否能吸引听众
- 是否有热情
- 是否有个人风格
- 听众是否有反应

---

### 3. 结构组织 (Structure)

权重：20%

#### 3.1 逻辑性 (Logic)
**评分范围**：0-10分

**评分标准**：
- **9-10分**：逻辑极为严密，推理完美
- **7-8分**：逻辑清晰，推理合理
- **5-6分**：逻辑基本清晰，有小瑕疵
- **3-4分**：逻辑不够清晰，有漏洞
- **0-2分**：逻辑混乱，完全没有条理

**检查要点**：
- 是否有明确的结构
- 论点是否有支撑
- 推理是否合理
- 是否有逻辑跳跃

#### 3.2 层次感 (Structure)
**评分范围**：0-10分

**评分标准**：
- **9-10分**：层次极为清晰，结构完美
- **7-8分**：层次清晰，结构良好
- **5-6分**：有基本结构，但层次不够分明
- **3-4分**：结构不够清晰，层次混乱
- **0-2分**：完全没有结构，一团乱麻

**检查要点**：
- 是否有开头、主体、结尾
- 部分划分是否清晰
- 过渡是否自然
- 是否有小结

#### 3.3 时间控制 (Time Management)
**评分范围**：0-10分

**评分标准**：
- **9-10分**：时间控制完美，完全符合预期
- **7-8分**：时间控制良好，基本符合预期
- **5-6分**：时间控制尚可，略有超时或不足
- **3-4分**：时间控制不好，超时或不足较多
- **0-2分**：时间完全失控

**检查要点**：
- 是否在预期时间内
- 各部分时间分配是否合理
- 关键内容时间是否充足
- 是否有充足的结尾时间

---

### 4. 观众互动 (Engagement)

权重：15%

#### 4.1 参与度 (Participation)
**评分范围**：0-10分

**评分标准**：
- **9-10分**：观众参与度极高，互动非常活跃
- **7-8分**：观众参与度高，有较多互动
- **5-6分**：有一定参与度，有少量互动
- **3-4分**：参与度较低，几乎没有互动
- **0-2分**：完全没有参与，观众被动

**检查要点**：
- 是否有提问
- 是否有回应
- 观众是否积极
- 是否有互动环节

#### 4.2 互动频率 (Frequency)
**评分范围**：0-10分

**评分标准**：
- **9-10分**：互动频率极高，恰到好处
- **7-8分**：互动频率良好，比较合适
- **5-6分**：有一定互动，频率尚可
- **3-4分**：互动较少，频率偏低
- **0-2分**：完全没有互动

**检查要点**：
- 互动是否自然
- 频率是否合适
- 是否打断演讲
- 是否能促进理解

#### 4.3 情感共鸣 (Empathy)
**评分范围**：0-10分

**评分标准**：
- **9-10分**：情感共鸣极强，观众深受感动
- **7-8分**：情感共鸣强，观众有反应
- **5-6分**：有一定情感共鸣
- **3-4分**：情感共鸣较弱
- **0-2分**：完全没有情感共鸣

**检查要点**：
- 观众是否有情感反应
- 是否能引起共鸣
- 是否有共情表达
- 情绪是否传递到位

---

### 5. 整体印象 (Overall Impression)

权重：10%

**评分范围**：0-10分

综合所有维度，给出整体印象分。

---

## 评分计算方法

### 综合评分公式

```
综合评分 = (
  内容质量 × 0.3 +
  表达能力 × 0.25 +
  结构组织 × 0.20 +
  观众互动 × 0.15 +
  整体印象 × 0.10
) × 10
```

### 子维度评分计算

每个主维度的评分为其下各子维度的平均分。

```
内容质量 = (表达清晰度 + 洞见深度 + 内容丰富度 + 可信度) / 4
表达能力 = (流畅度 + 语速控制 + 语气表达 + 感染力) / 4
结构组织 = (逻辑性 + 层次感 + 时间控制) / 3
观众互动 = (参与度 + 互动频率 + 情感共鸣) / 3
```

## 评价内容生成指南

### 评价开头（10-15秒）

使用预制开头词库，示例：

- "感谢{speaker}的精彩分享！"
- "非常感谢{speaker}的演讲！"
- "{speaker}刚才的分享太精彩了！"
- "感谢{speaker}给我们带来这么好的内容！"

### 评价主体（20-30秒）

#### 结构
1. **亮点突出**（1-2个最突出的亮点）
2. **洞见补充**（1个有深度的见解）
3. **建议**（1个建设性建议，可选）

#### 示例
```
最让我印象深刻的是内容的洞见深度，特别是提出的深度学习可以提高诊断准确率的观点，让我们对AI医疗有了全新的认识。整场演讲逻辑清晰、数据充分、情感真挚，是一场高质量的演讲！
```

### 评价结尾（5-10秒）

简洁收尾，示例：

- "期待更多精彩分享！"
- "谢谢！"
- "非常棒！"

---

## 评价质量检查清单

- [ ] 评价开头已使用预制词库
- [ ] 评价开头时长10-15秒
- [ ] 评价主体突出了1-2个亮点
- [ ] 评价主体补充了1个洞见
- [ ] 评价主体时长20-30秒
- [ ] 评价有简洁结尾
- [ ] 整体评价时长30-45秒
- [ ] 评价以赞扬为主，建议为辅
- [ ] 评价客观公正，不过分吹捧
- [ ] 评价语言自然流畅

## 评价模板

### 优秀演讲（8-10分）

```
感谢{speaker}的精彩分享！
最让我印象深刻的是{亮点1}，特别是{具体例子}，让我们{收获}。还有{亮点2}也非常精彩！整场演讲{总体评价}，是一场高质量的演讲！
期待更多精彩分享！
```

### 良好演讲（6-8分）

```
非常感谢{speaker}的演讲！
{亮点1}让人印象深刻，{亮点2}也很有价值。总体来说，演讲{总体评价}，做得很好！
期待更多分享！
```

### 一般演讲（4-6分）

```
感谢{speaker}的分享！
{亮点1}还不错，如果能{改进建议}会更好。总体来说，演讲{总体评价}。
谢谢！
```
